**QuickScrim**
 
 QuickScrim is a web application that lets users:
 
 * Register for an account
 * Log into account
 * Edit your account
 * Create and view events
 * Edit/delete your events
 * Post on a sport forum
 * Edit/delete your posts
 
 Technologies used for this application:
 
 * Spring Boot
 * Thymeleaf
 * MySQL
 * CSS
 * BootStrap
